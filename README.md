# DT-AKT
Robust feature extraction for visible-NIR image registration through unsupervised dual training with adaptive knowledge transfer

This repository provides the official **TensorFlow/Keras** implementation of the model architecture and inference logic for the paper:  
**"Robust Feature Extraction for Visible-NIR Image Registration through Unsupervised Dual Training with Adaptive Knowledge Transfer"**

[![Paper](https://img.shields.io/badge/Paper-PDF-red)](https://doi.org/10.1016/j.patcog.2026.113814)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## 📖 Overview
This repository contains the core **DT-AKT** structure as proposed in our research. The model focuses on aligning visible and near-infrared images.

While the GitHub link was not present in the original publication, this is the official implementation intended to assist researchers in reproducing the architecture and integrating it into their own TensorFlow-based workflows.

## 🚀 Key Features
* **Keras Functional/Subclassing API**: Implementation using standard TensorFlow 2.10 patterns.
* **Architecture Definition**: Full transparency of the layers, connections, and hyperparameters used in the paper.
* **Ready-to-Use**: Minimal setup required to initialize the model for testing or transfer learning.
